# Controls — Dudes in Alaska

This document lists every input binding in the game.
All actions are defined in **Project → Project Settings → Input Map** and support both keyboard/mouse and gamepad.

---

## Movement

| Action | Keyboard | Gamepad |
|--------|----------|---------|
| Move left | **A** | Left Stick ← |
| Move right | **D** | Left Stick → |
| Move up | **W** | Left Stick ↑ |
| Move down | **S** | Left Stick ↓ |

---

## Actions

| Action | Keyboard | Gamepad |
|--------|----------|---------|
| Interact | **E** | A (×) |
| Check Needs | **T** | Y (△) |
| Open Inventory | **I** | X (□) |
| Open Journal | **J** | — |
| Open Map | **M** | — |
| Pause | **Esc** | Start / Menu |

---

## Touch (Mobile / Web)

On touch devices a virtual joystick appears on the left half of the screen and action buttons appear in the bottom-right corner.

| On-screen control | Action |
|-------------------|--------|
| Left-half drag | Move |
| **E** button | Interact |
| **T** button | Check Needs |
| **\|\|** button | Pause |

---

## Controls Overlay

An input reference overlay is shown automatically when a level starts.
**Click, tap, or press any key** to dismiss it.
