# Dudes in Alaska

A 2D game built with [Godot 4](https://godotengine.org/).

## Getting Started

1. Download [Godot 4](https://godotengine.org/download) (Standard, no Mono)
2. Open Godot and import `project.godot` from this repository
3. Press **F5** to run
